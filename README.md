## Estrutura do Repositorio

O projeto esta organizado por areas de estudo, focando em codigos praticos e sistemas completos:

- [android/task-hero](./android/task-hero) -- TaskHero: Aplicativo mobile em Kotlin com Jetpack Compose e Firebase.
- [backend/shopflow-api](./backend/shopflow-api) -- ShopFlow-API: API de E-commerce escalavel em Node.js/TypeScript com JWT e PostgreSQL.
- [estrutura-de-dados](./estrutura-de-dados) -- Implementacoes de Arvores (Binaria, Busca, AVL) e Tabelas Hash em C.
- [java](./java) -- Algoritmos de ordenacao e o sistema de Menu Escolar (Array vs LinkedList).
- [java-fx](./java-fx) -- Aplicacoes Desktop com interface grafica, utilizando Maven, Hibernate e JPA (Sistema Academico, Biblioteca, etc).

## Tecnologias Utilizadas

- **Node.js/TypeScript** -- Express, TypeORM, JWT, Yup, Nodemailer
- **Android/Kotlin** -- Jetpack Compose, ViewModels, Firebase Firestore/Auth
- **Java** -- POO, algoritmos, JavaFX 
- **C** -- Estrutura de Dados  
- **JavaFX** -- Interfaces graficas e aplicacoes desktop 
- **SQL** -- Criacao e manipulacao de bancos de dados relacionais (MySQL)
